# CREATIVE SPEC

You are an elite creative director and front-end developer. Build a single-file website based on the theme below. This is a test of pure creativity.

**Theme:** [INSERT THEME HERE]

**Requirements:**
- One `index.html` file
- Mobile-first — **strict 9:16 portrait aspect ratio** (game UI must fit and function in portrait)
- Vanilla code (Three.js allowed)
- GitHub Pages compatible
- Minimum 500 lines of real code

**You MUST include a playable mini-game as the primary mechanic** — the game is not optional. In addition to the game, include at least **one more** of these:
- 3D scene
- Particle system or generative canvas
- Scroll or interaction animations (touch/drag)
- Physics or drag interactions
- Morphing, distortion, or liquid effects
- Explosions or particle bursts

**Theme expression:** The theme is not a label. Every visual, game mechanic, color, shape, motion, and atmosphere must clearly communicate the theme. If the theme word itself was removed, a viewer should correctly guess the theme within 2 seconds of seeing the game.

**Usability:** The user must understand **how to play the game** within 1 second of looking. Show the goal clearly (score, timer, or objective). Make tappable areas obvious. If the game uses touch/swipe/tilt, indicate that. Do not hide interactions. Clarity and spectacle must coexist.

**Do not:**
- Use centered cards
- Use basic fade transitions
- Build a generic landing page
- State the theme as plain text and stop there
- Make the game secondary or hidden

**Failure conditions:**
- Under 500 lines
- No playable mini-game
- Game is broken or unclear
- Theme is only visible as text
- User cannot figure out how to play in 1 second
- Game designed for landscape (must work in portrait)
